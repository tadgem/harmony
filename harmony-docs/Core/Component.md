A `Component` can be anything that we wish to attach to an entity, there is no Type restriction, more a guiding principle that Components should only contain data. This is a bit nebulous so an example would be something like a physics Rigidbody. The component should have a reference to the Rigidbody object, but should not contain a reference to the physics world, as it is not a responsibility of the Rigidbody `Component`, it is a responsibility of the `System`
